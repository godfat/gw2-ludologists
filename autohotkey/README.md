# [AutoHotkey](https://autohotkey.com/) scripts for Guild Wars 2

## dodge-jump.ahk

Press <kbd>Shift</kbd> + <kbd>Space</kbd> to dodge jump.

## capslock.ahk

Press <kbd>CapsLock</kbd> to press <kbd>N</kbd> (default hotkey for special
action)

## auto-click.ahk

Press <kbd>F9</kbd> to toggle auto-click at current mouse position.

## mystic-forge.ahk

Press <kbd>F10</kbd> to record the cursor position for <kbd>Refill</kbd>, and
press <kbd>F11</kbd> to record the cursor position for <kbd>Forge</kbd>,
finally press <kbd>F12</kbd> to start running it for 50 times.

Press <kbd>F12</kbd> again during the process to stop it early.
